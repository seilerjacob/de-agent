# Python Skill: Data Quality

Demonstrates a lightweight data quality validation framework that can be
embedded at any stage of an ETL pipeline.

## What This Covers
- **Expectation-based checks** (inspired by Great Expectations (GX))
- Null, uniqueness, range, and custom checks
- Structured result reporting (pass/fail per check)
- Configurable fail-fast vs. collect-all modes

## File
- `quality_checks.py` — Reusable data quality check framework
