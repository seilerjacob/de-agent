# Python Skill: Data Transforms

Demonstrates common data transformation patterns used in Data Engineering (DE)
pipelines, using both pandas and polars.

## What This Covers
- **Pandas transforms** — cleaning, pivoting, type casting, derived columns
- **Polars transforms** — lazy evaluation, expression API, high-performance joins
- Side-by-side comparison of the two libraries

## File
- `transforms.py` — Transform functions for both pandas and polars
