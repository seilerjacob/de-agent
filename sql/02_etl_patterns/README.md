# SQL Skill: ETL Patterns

Demonstrates common Extract-Transform-Load (ETL) patterns written in pure SQL.

## What This Covers
- **Incremental load** using a high-water mark
- **SCD Type 2** merge pattern for historical dimension tracking

## Files
- `incremental_load.sql` — High-water mark incremental load with MERGE
- `scd_type2.sql` — SCD Type 2 implementation using MERGE
