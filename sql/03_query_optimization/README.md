# SQL Skill: Query Optimization

Demonstrates advanced SQL patterns for readable, performant analytical queries.

## What This Covers
- **Window functions** — ranking, running totals, moving averages
- **Common Table Expression (CTE) patterns** — recursive CTEs, modular query design

## Files
- `window_functions.sql` — Practical window function examples
- `cte_patterns.sql` — CTE-based query decomposition and recursion
