# SQL Skill: Data Quality

Demonstrates SQL-based data quality validation checks that can be embedded in
ETL pipelines or run as standalone audits.

## What This Covers
- **Null checks** on required columns
- **Uniqueness** validation
- **Referential integrity** verification
- **Range and format** validation
- **Row count anomaly detection** vs. historical averages

## File
- `validation_checks.sql` — A suite of reusable data quality queries
